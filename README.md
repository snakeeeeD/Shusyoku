# Think or Sink（仮）

## 概要
ローグライク・デッキ構築ゲーム。
個人開発・制作中。

## 自分の担当範囲
企画・設計・実装まで全て個人で担当。
ローグライク・デッキ構築システムを自作エンジン上に構築している。

ImGui（オープンソースライブラリ）をデバッグ用UIに使用しているが、
ゲームロジック・描画処理・カード効果システム等は全て自身の実装。

## 使用技術
DirectX11 / C++ / 自作エンジン / ImGui（デバッグUI）

## 工夫した点・課題と解決
カードの効果処理を、メイン効果とヒット時効果を分離した構造に再設計し、
拡張しやすい仕組みに整理した。可変するグリッド形状を持つボスや、
データ駆動のデバフ処理など、プレイヤーに常に考えさせる仕掛けを増やせる
設計を意識している。最初から完璧な設計を目指すより、動くものを作って
必要になってから抽象化するスタイルで進めている。

## アピールポイント
「Slay the Spire」をプレイして受けた感動と興奮を、自分の手で誰かに
届けたいという思いから着想。憧れのゲーム、憧れの業界に挑戦する一歩として
開発を続けている。「プレイヤーに常に考えさせたい」という軸を持ちながら、
設計から実装までを見直し続けている。
